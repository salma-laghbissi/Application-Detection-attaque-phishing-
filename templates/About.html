Les algorithmes ML utilisés

L'arbre de décision : Algorithme d'apprentissage automatique pour la classification, choisissant les caractéristiques les plus importantes à chaque étape pour des décisions facilement interprétables.

Random Forest : Algorithme de classification utilisant de multiples arbres de décision sur des sous-ensembles aléatoires de données, pour une prédiction robuste et moins sujette au surapprentissage.

XGBoost : Algorithme de classification utilisant une série de modèles d'arbres de décision successifs, optimisant l'erreur de prédiction à chaque étape pour améliorer les performances globales du modèle.

Support Vector Machine : Algorithme de classification trouvant l'hyperplan optimal séparant les données en différentes classes, maximisant la marge entre les points de données les plus proches de chaque classe.

Régression Logistique : Algorithme de classification estimant les probabilités de classe à l'aide d'une fonction logistique et d'une fonction de seuil pour déterminer la classe prédite.
